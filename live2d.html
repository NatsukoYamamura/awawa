标准化工作流程

委托前

  • 客户提供：分层清晰 PSD + 参考视频 + 需求清单
  • 模师评估：确认复杂度 → 正式报价 → 平台下单 → 排期

制作中（分阶段确认）

  • Q 版：草图绑定 → 物理调试 → 成品预览（2-3 轮确认）
  • 正比：面部 → 身体 → 物理 → 动画 → 成品（4-5 轮确认）
  • 修改规则：每阶段限 2 次免费修改，超出按 30-50 元/次

交付后

  • 交付内容：.moc3 + .json + VTS 预设 + 使用说明书
  • 免费维护：30 天内修复非人为技术问题
  • 工程文件：默认不提供，买断需提前约定
